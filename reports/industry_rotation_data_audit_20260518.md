## 5. 行业组合构建方式

```
月末 T 日:
  1. 取最近季度 HS300 成分股列表 (historical_constituents.json)
  2. 从 HS300_daily 过滤，剔除 < 3 股的行业
  3. 主方案(等权): mean(close.pct_change()) per industry
  4. 对照方案(成交额加权): sum(ret * amount) / sum(amount) per industry
  5. 月末 rank 行业过去 N 日收益 → top-3 / top-5
  6. T+1 open 建仓，持有至下月末
```

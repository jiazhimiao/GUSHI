# 趋势突破策略 - 优化方向

## 当前状态

策略核心逻辑正确（右侧交易 + 空仓保护），但存在过度交易、假突破多、单边行情依赖等问题。

---

## 一、精选入场（减少假突破）

| 优化点 | 当前 | 建议 | 预期效果 |
|--------|------|------|----------|
| 突破确认 | 单日突破即入场 | 等次日确认（收盘不跌回突破价下方） | 过滤15-25%假突破 |
| 放量标准 | 固定1.5倍 | 用ATR自适应：高波动股要求更高放量倍数 | 减少低质量信号 |
| 趋势强度 | 无 | 加ADX>20过滤，只交易有方向的股票 | 过滤震荡区假突破 |
| 缺口过滤 | 无 | 跳空>3%不追，等回踩 | 避免追高被套 |
| 多周期确认 | 仅日线 | 周线趋势向上时，日线突破才有效 | 提高胜率 |

## 二、智能止盈止损

| 优化点 | 当前 | 建议 | 预期效果 |
|--------|------|------|----------|
| 止损方式 | 固定8% | ATR自适应止损（2倍ATR），波动大放宽、波动小收紧 | 适应不同股票特征 |
| 移动止盈 | 无 | 盈利>15%后将止损上移到成本价 | 保护利润 |
| 时间止损 | 无 | 持仓N天不涨反跌则主动退出 | 释放资金给更好的机会 |
| 分批止盈 | 无 | 盈利20%卖一半，剩下用均线跟踪 | 吃主升浪尾部 |

## 三、仓位管理

| 优化点 | 当前 | 建议 | 预期效果 |
|--------|------|------|----------|
| 单票仓位 | 固定等权 | 波动率倒数加权（高波动=小仓位） | 组合更稳定 |
| 板块上限 | 无 | 同行业最多3只，避免集中 | 降低板块黑天鹅风险 |
| 市场环境适配 | 简单二分（广度>40%全仓，<40%空仓） | 三档：>50%全仓、30-50%半仓、<30%空仓 | 过渡期更平滑 |

## 四、数据处理

| 优化点 | 当前 | 建议 | 预期效果 |
|--------|------|------|----------|
| 成分股偏差 | 用当前成分股回测全时段 | 每年初重新获取成分股列表 | 消除幸存者偏差 |
| 财报日期 | 未考虑 | 财报发布前后3天不交易 | 避免业绩雷 |
| 流动性过滤 | 仅过滤成交额<5000万 | 加日均成交额排名（取前80%），剔除流动性差的 | 减少滑点 |

## 五、组合层面

| 优化点 | 当前 | 建议 | 预期效果 |
|--------|------|------|----------|
| 相关性控制 | 无 | 持股间相关性>0.7时只保留更强的 | 真正分散 |
| 最大回撤熔断 | 无 | 策略回撤>15%暂停1个月，等市场稳定 | 防止连续亏损 |

---

## 优先级建议

**第一优先（立刻做，效果明显）：**
1. ATR自适应止损 → 降低回撤
2. 时间止损 + 移动止盈 → 释放无效资金
3. 三档市场环境 → 不再简单空仓/满仓

**第二优先（数据质量）：**
4. 历史成分股 → 消除幸存者偏差
5. 财报过滤 → 避雷

**第三优先（精进）：**
6. 波动率加权仓位
7. 多周期确认
